#Installation Instructions

1. Go to https://github.com/eerohelin/Better-LOLPros
2. Click on "Code"
3. Download Zip
4. Unzip the file and follow these instructions:
Visit chrome://extensions
Enable Developer mode by ticking the checkbox in the upper-right corner
Click on the "Load unpacked extension..." button
Select the unzipped folder
(If you run into an error loading the folder, make sure you are loading the unzipped folder and not the folder CONTAINING the folder)
And with that you should have the extension up and running!
